## Somador de 4 bits

&nbsp;&nbsp;&nbsp;&nbsp;Para esta atividade, foi desenvolvido um circuito lógico de um somador de 4 bits.

### Ferramentas

- [Digital](https://github.com/hneemann/Digital).
